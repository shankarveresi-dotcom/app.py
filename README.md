# 📄 AI Resume Analyzer

An intelligent web application that analyzes a candidate's resume against a job description, calculates a skill-matching score, identifies matched and missing skills, and provides improvement suggestions.

## 🚀 Features

* Upload resumes in PDF format
* Automatically extract text from resumes
* Identify technical skills
* Compare resume skills with job requirements
* Calculate a resume-job match percentage
* Display matched skills
* Identify missing skills
* Provide personalized improvement suggestions
* Interactive web interface built with Streamlit

## 🛠️ Technologies Used

* Python
* Streamlit
* PyPDF2
* Regular Expressions
* Natural Language Processing concepts

## 🎯 How It Works

1. The user uploads a resume in PDF format.
2. The application extracts text from the uploaded resume.
3. The user pastes a job description.
4. Technical skills are identified from both the resume and job description.
5. The application compares the skills and calculates a matching percentage.
6. The dashboard displays matched skills, missing skills, and improvement suggestions.

## ⚙️ Installation

Clone the repository:

git clone YOUR_REPOSITORY_URL

Navigate to the project folder:

cd AI-Resume-Analyzer

Install the required dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
